# homepage
The Solid Network Bank Protected Page

## Create React App

### To create a new app.

```sh
 npx create-react-app protectedpage
```

### Starts the development server.

 ```sh   
npm start
```

### Bundles the app into static files for dev.

```sh
npm run build:dev
```

### Starts the test runner.

```sh
npm test
```

## Make React API Calls

```sh
npm install axios
```
